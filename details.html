Introduction

The Scalable Thread Management Library is a C-based project designed to manage threads efficiently using a thread pool, priority-based task scheduling, and performance monitoring. It leverages POSIX threads (pthreads) to handle dynamic thread creation, task execution, and resource optimization. This project was developed collaboratively by a team as part of a programming assignment.

File Structure

  • Thread.c, Thread.h: Basic thread creation test.
  • MessageQueue.c, MessageQueue.h: Linked-list-based task queue with priority and timeout.
  • Performance.c, Performance.h: Performance metrics (execution time, throughput).
  • thread_manager.c: Main driver with thread pool, task execution, and performance tracking.
